javaScript基础学习javaScript中的键盘事件

-2023年11月8号

标签：   前端     javaScript     代码

佛教中有一句话：初学者的心态；拥有初学者的心态是件了不起的事情。

出自乔布斯

   在网页开发中，经常要捕捉键盘的事件，例如按下Enter完成表单提交，按下Backspaccce关闭项目，按下 Esc关闭项目，使用
javaScript中的键盘监控事件就可以完成这种功能，语法如下：

document.onkeydown=function keydown(){}

   以下是通过监控Enter键，提交表单的示例：

						function transitionEnd() {
						  var el = document.createElement('bootstrap')
						
						  var transEndEventNames = {
						    WebkitTransition : 'webkitTransitionEnd',
						    MozTransition    : 'transitionend',
						    OTransition      : 'oTransitionEnd otransitionend',
						    transition       : 'transitionend'
						  }
						
						  for (var name in transEndEventNames) {
						    if (el.style[name] !== undefined) {
						      return { end: transEndEventNames[name] }
						    }
						  }
						
						  return false // explicit for ie8 (  ._.)
						}
						
						// https://blog.alexmaccaw.com/css-transitions
						$.fn.emulateTransitionEnd = function (duration) {
						  var called = false
						  var $el = this
						  $(this).one('bsTransitionEnd', function () { called = true })
						  var callback = function () { if (!called) $($el).trigger($.support.transition.end) }
						  setTimeout(callback, duration)
						  return this
						}

keydown事件和键位码的解释如下所示：

keydown事件      当访客用鼠标选中了某个节点，按下键盘即会触发事件

键盘键位码      使用十进制给键盘上每个键进行了编号，例如回车的键位码是13，空格的键位码是32
